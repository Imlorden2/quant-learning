## 🏷️ 价格和收益相关术语

### 基础价格术语

**开盘价 (Open Price)**
- 定义：交易日开市时第一笔成交的价格
- 英文：Open
- 意义：反映隔夜情绪和消息面影响
- 例子：昨日收盘10元，今日开盘10.5元，说明有利好消息

**收盘价 (Close Price)**  
- 定义：交易日收市时最后一笔成交的价格
- 英文：Close
- 意义：当日交易的最终结果，影响次日开盘
- 注意：A股收盘价是最后3分钟成交量加权平均价

**最高价/最低价 (High/Low)**
- 定义：交易日内达到的最高/最低成交价
- 意义：反映当日波动幅度和支撑阻力
- 应用：计算真实波动率ATR

**均价 (VWAP - Volume Weighted Average Price)**
- 定义：成交量加权平均价格
- 计算：VWAP = Σ(价格×成交量) / Σ成交量
- 用途：机构交易执行基准，算法交易目标

### 涨跌相关

**涨跌幅 (Price Change %)**
- 计算：涨跌幅 = (收盘价 - 昨收价) / 昨收价 × 100%
- A股限制：主板±10%，创业板/科创板±20%
- ST股票：±5%

**涨跌停 (Price Limit)**
- 涨停：股价达到当日涨幅上限，买盘强烈
- 跌停：股价达到当日跌幅下限，卖盘汹涌
- 一字涨停/跌停：全天都在涨停/跌停价位

**振幅 (Amplitude)**
- 计算：振幅 = (最高价 - 最低价) / 昨收价 × 100%
- 意义：反映股价波动的激烈程度
- 应用：识别放量突破或诱多诱空

### 复权相关

**前复权 (Forward Adjusted)**
- 定义：以当前价格为基准，调整历史价格
- 公式：前复权价 = 历史价格 × 复权因子
- 优点：当前价格真实，适合技术分析

**后复权 (Backward Adjusted)**
- 定义：以上市价为基准，调整当前价格
- 优点：显示真实投资收益
- 缺点：当前价格不是真实价格

**除权除息 (Ex-rights and Ex-dividend)**
- 除权日：分红配股权利登记的最后一天
- 影响：价格会相应调整下降
- 填权：股价重新上涨至除权前水平

## 📈 交易机制术语

### 基础交易术语

**做多/买涨 (Long Position)**
- 定义：买入资产，期望价格上涨获利
- 盈利：买低卖高
- 风险：价格下跌时亏损

**做空/买跌 (Short Position)**
- 定义：卖出资产，期望价格下跌获利
- A股限制：只能通过融券、股指期货做空
- 盈利：卖高买低
- 风险：理论上无限损失

**多头/空头 (Bull/Bear)**
- 多头：看涨的投资者，持有多仓
- 空头：看跌的投资者，持有空仓
- 多空比：反映市场情绪的指标

**杠杆 (Leverage)**
- 定义：用借入资金扩大交易规模
- 计算：杠杆比例 = 总资产 / 自有资金
- 例子：100万自有资金 + 100万借款 = 2倍杠杆
- 影响：放大收益和亏损

### 高级交易概念

**滑点 (Slippage)**
- 定义：实际成交价与预期价格的差异
- 成因：
  - 市场冲击：大单影响价格
  - 流动性不足：买卖盘稀薄
  - 网络延迟：价格变化太快
- 控制：分单交易、选择流动性好的时段

**市场冲击 (Market Impact)**
- 定义：交易行为对价格造成的影响
- 永久冲击：价格永久偏离
- 临时冲击：交易完成后价格回归
- 量化公式：冲击成本 ∝ √交易量

**流动性 (Liquidity)**
- 定义：资产快速买卖而不显著影响价格的能力
- 衡量指标：
  - 买卖价差：越小越好
  - 成交量：越大越好
  - 盘口深度：挂单量越多越好

## ⚠️ 风险管理术语

### 基础风险概念

**止损 (Stop Loss)**
- 定义：设定亏损上限，触及即平仓
- 类型：
  - 固定止损：-5%止损
  - 跟踪止损：随价格调整的动态止损
  - 时间止损：持有时间超限止损

**止盈 (Take Profit)**
- 定义：设定盈利目标，达到即平仓
- 策略：
  - 固定目标：+10%止盈
  - 分批止盈：50%仓位+5%，剩余+10%
  - 移动止盈：盈利后调整止损线

**仓位控制 (Position Sizing)**
- 定义：决定每次交易投入资金的比例
- 方法：
  - 固定比例：每次10%资金
  - 等风险：每次承担相同风险
  - 凯利公式：Kelly% = (bp-q)/b

**资金管理 (Money Management)**
- 核心原则：
  - 不要把鸡蛋放在一个篮子里
  - 控制单笔损失在总资金的1-2%
  - 保持足够现金应对机会

### 高级风险概念

**回撤 (Drawdown)**
- 定义：从高点到低点的损失百分比
- 最大回撤：历史上最大的回撤幅度
- 计算：回撤 = (峰值 - 谷值) / 峰值 × 100%
- 重要性：比收益率更重要的风险指标

**风险价值 (VaR - Value at Risk)**
- 定义：在给定置信水平下，预期的最大损失
- 例子：95% VaR = 100万，意味着95%概率损失不超过100万
- 计算方法：历史模拟法、方差-协方差法、蒙特卡洛法

**风险平价 (Risk Parity)**
- 定义：各资产对组合风险贡献相等
- 传统组合：按资金配置
- 风险平价：按风险配置
- 优势：更均衡的风险分散

## 📊 技术分析术语

### 趋势相关

**趋势 (Trend)**
- 上升趋势：高点和低点依次抬高
- 下降趋势：高点和低点依次降低
- 横盘整理：在一定区间内震荡

**趋势线 (Trend Line)**
- 上升趋势线：连接两个或多个低点
- 下降趋势线：连接两个或多个高点
- 突破：价格穿越趋势线，可能趋势反转

**支撑阻力 (Support/Resistance)**
- 支撑位：价格下跌时遇到买盘支撑的价位
- 阻力位：价格上涨时遇到卖盘阻力的价位
- 角色转换：突破阻力位后变为支撑位

### 技术指标

**移动平均线 (Moving Average, MA)**
- 简单移动平均：SMA = (C1+C2+...+Cn)/n
- 指数移动平均：EMA，近期价格权重更大
- 金叉：短期MA上穿长期MA，买入信号
- 死叉：短期MA下穿长期MA，卖出信号

**相对强弱指数 (RSI)**
- 计算：RSI = 100 - 100/(1+RS)，RS = 平均涨幅/平均跌幅
- 超买：RSI > 70
- 超卖：RSI < 30
- 背离：价格新高而RSI不创新高

**MACD**
- 组成：DIF线、DEA线、MACD柱
- DIF = EMA(12) - EMA(26)
- DEA = EMA(DIF, 9)
- MACD = 2×(DIF - DEA)

**布林带 (Bollinger Bands)**
- 中轨：20日移动平均线
- 上轨：中轨 + 2×标准差
- 下轨：中轨 - 2×标准差
- 应用：价格触及上轨卖出，触及下轨买入

## 🤖 量化策略术语

### 策略分类

**因子 (Factor)**
- 定义：影响股票收益的变量
- 分类：
  - 价值因子：PE、PB、股息率
  - 成长因子：ROE、营收增长率
  - 质量因子：负债率、盈利稳定性
  - 动量因子：过去收益率、价格趋势

**阿尔法 (Alpha)**
- 定义：策略超越基准的超额收益
- 计算：Alpha = 策略收益 - 基准收益
- 目标：获得正Alpha

**贝塔 (Beta)**
- 定义：策略相对市场的敏感度
- 计算：Beta = Cov(策略,市场) / Var(市场)
- 解释：Beta = 1.5，市场涨10%，策略涨15%

**多空策略 (Long-Short)**
- 做多：买入被低估的股票
- 做空：卖出被高估的股票
- 优势：市场中性，降低系统性风险

### 策略类型

**动量策略 (Momentum)**
- 逻辑：强者恒强，弱者恒弱
- 实现：买入近期表现好的股票
- 风险：趋势反转风险

**均值回归策略 (Mean Reversion)**
- 逻辑：价格偏离均值后会回归
- 实现：买入被超卖的股票
- 风险：价值陷阱风险

**配对交易 (Pairs Trading)**
- 逻辑：历史相关的两只股票价差会回归
- 实现：价差扩大时，买入便宜的，卖出贵的
- 优势：市场中性策略

**套利策略 (Arbitrage)**
- 无风险套利：同一资产在不同市场的价差
- 统计套利：基于统计规律的套利
- 风险：价差不收敛风险

## 📈 绩效评估术语

### 收益指标

**年化收益率 (Annualized Return)**
- 计算：(期末净值/期初净值)^(365/持有天数) - 1
- 意义：便于不同期限策略比较
- 注意：复利计算

**累计收益率 (Cumulative Return)**
- 计算：期末净值/期初净值 - 1
- 意义：投资期间总收益
- 特点：不考虑时间因素

**超额收益 (Excess Return)**
- 计算：策略收益 - 基准收益
- 基准选择：沪深300、中证500等
- 意义：策略的真实贡献

### 风险调整指标

**夏普比率 (Sharpe Ratio)**
- 计算：(年化收益 - 无风险利率) / 年化波动率
- 意义：每承受一单位风险的超额收益
- 优秀标准：> 1

**索提诺比率 (Sortino Ratio)**
- 计算：(年化收益 - 目标收益) / 下行标准差
- 改进：只考虑下行波动，更合理
- 应用：更适合评估绝对收益策略

**卡玛比率 (Calmar Ratio)**
- 计算：年化收益 / 最大回撤
- 意义：单位回撤风险的收益
- 优势：直观反映风险收益比

**信息比率 (Information Ratio)**
- 计算：Alpha / 跟踪误差
- 意义：获得超额收益的稳定性
- 应用：评估主动管理能力

### 其他重要指标

**胜率 (Win Rate)**
- 计算：盈利交易次数 / 总交易次数
- 意义：策略成功概率
- 注意：高胜率不代表高收益

**盈亏比 (Profit/Loss Ratio)**
- 计算：平均盈利 / 平均亏损
- 意义：单次交易的风险收益比
- 目标：> 1

**最大回撤期 (Maximum Drawdown Duration)**
- 定义：从高点到新高点的最长时间
- 意义：心理承受力测试
- 重要性：比最大回撤幅度更煎熬

## 🏢 市场结构术语

### 参与者分类

**机构投资者 (Institutional Investor)**
- 包括：基金、保险、社保、QFII
- 特点：资金量大、投资期限长、行为理性
- 影响：主导长期趋势

**散户投资者 (Retail Investor)**
- 特点：资金量小、情绪化交易
- 行为：追涨杀跌、频繁交易
- 作用：提供流动性、制造波动

**量化交易者 (Quantitative Trader)**
- 特点：程序化交易、追求统计优势
- 策略：高频交易、套利、因子投资
- 影响：提高市场效率、降低价差

### 交易方式

**算法交易 (Algorithmic Trading)**
- TWAP：时间加权平均价格算法
- VWAP：成交量加权平均价格算法
- Implementation Shortfall：执行缺口算法
- 目的：降低冲击成本、隐藏交易意图

**高频交易 (High-Frequency Trading)**
- 特点：毫秒级决策、大量小额交易
- 策略：做市、套利、动量
- 优势：速度快、成本低
- 争议：是否增加市场不稳定性

**暗池交易 (Dark Pool)**
- 定义：不公开报价和成交信息的交易平台
- 优势：大宗交易不暴露意图
- 风险：价格发现功能削弱

## 🔄 衍生品术语

### 期货相关

**期货合约 (Futures Contract)**
- 定义：约定未来某时点买卖资产的标准化合约
- 特点：杠杆交易、双向交易、每日结算
- 应用：对冲、投机、套利

**基差 (Basis)**
- 定义：现货价格 - 期货价格
- 正基差：现货价格 > 期货价格
- 负基差：现货价格 < 期货价格
- 应用：期现套利策略

**移仓 (Roll Over)**
- 定义：平仓即将到期的合约，开仓远月合约
- 原因：避免实物交割
- 成本：远近月价差形成的移仓成本

### 期权相关

**看涨期权 (Call Option)**
- 定义：在到期日以执行价买入标的的权利
- 盈利：标的价格 > 执行价 + 期权费
- 最大损失：期权费

**看跌期权 (Put Option)**
- 定义：在到期日以执行价卖出标的的权利
- 盈利：标的价格 < 执行价 - 期权费
- 应用：保险、对冲

**隐含波动率 (Implied Volatility)**
- 定义：期权价格反映的标的未来波动率预期
- 应用：评估期权是否高估或低估
- 波动率交易：买入低IV期权，卖出高IV期权

**希腊字母 (Greeks)**
- Delta：期权价格对标的价格的敏感度
- Gamma：Delta的变化率
- Theta：期权的时间价值衰减
- Vega：期权价格对波动率的敏感度
