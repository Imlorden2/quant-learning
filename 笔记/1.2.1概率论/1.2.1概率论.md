## 🔢 基础概率概念

### 1. 概率的定义

**古典概率（等可能性）**
```
P(A) = 有利结果数 / 总结果数

例子：掷硬币
P(正面) = 1/2 = 0.5
```

**频率概率（大数定律）**
```
P(A) = lim(n→∞) 实验中A发生次数/总实验次数

例子：某策略
历史测试1000次，盈利650次
P(盈利) ≈ 650/1000 = 0.65
```

**主观概率（贝叶斯）**
```
基于已有信息和经验的主观判断

例子：分析师判断
"基于当前信息，我认为明天上涨概率80%"
```

### 2. 概率的基本性质

**公理化定义**
1. **非负性**: 0 ≤ P(A) ≤ 1
2. **规范性**: P(Ω) = 1 （必然事件概率为1）
3. **可加性**: P(A∪B) = P(A) + P(B) - P(A∩B)

**重要公式**
```python
# 互补事件
P(A') = 1 - P(A)

# 条件概率
P(A|B) = P(A∩B) / P(B)

# 全概率公式  
P(A) = Σ P(A|Bi) × P(Bi)

# 乘法定理
P(A∩B) = P(A|B) × P(B)
```

### 3. 独立性与相关性

**独立事件**
```python
# 定义：P(A∩B) = P(A) × P(B)
# 条件：P(A|B) = P(A)

例子：
连续两次抛硬币的结果相互独立
P(第二次正面|第一次正面) = P(第二次正面) = 0.5

# 量化交易中的误区
❌ 错误：连续5次止损，第6次一定盈利（赌徒谬误）
✅ 正确：如果交易相互独立，第6次盈利概率不变
```

**相关性**
```python
# 正相关：P(A|B) > P(A)
# 负相关：P(A|B) < P(A) 
# 不相关：P(A|B) = P(A)

市场例子：
- 银行股与地产股：正相关（利率敏感）
- 黄金与美元：负相关（避险属性）
- A股与原油价格：相关性较弱
```

## 📊 常见概率分布

### 1. 离散分布

#### 伯努利分布（Bernoulli）
**定义**：只有两个可能结果的实验
```python
# 参数：p（成功概率）
# 取值：X = 1（成功），X = 0（失败）
# 概率质量函数：
P(X = 1) = p
P(X = 0) = 1-p

# 期望：E(X) = p  
# 方差：Var(X) = p(1-p)

# 量化应用：单次交易盈亏
某策略胜率60%，单次交易：
P(盈利) = 0.6, P(亏损) = 0.4
```

#### 二项分布（Binomial）
**定义**：n次独立伯努利试验中成功的次数
```python
# 参数：n（试验次数），p（单次成功概率）
# 概率质量函数：
P(X = k) = C(n,k) × p^k × (1-p)^(n-k)

# 期望：E(X) = np
# 方差：Var(X) = np(1-p)

# 量化应用：100次交易中盈利次数
n = 100, p = 0.6
E(盈利次数) = 100 × 0.6 = 60次
标准差 = √(100×0.6×0.4) = √24 ≈ 4.9
```

#### 泊松分布（Poisson）  
**定义**：单位时间内随机事件发生的次数
```python
# 参数：λ（平均发生率）
# 概率质量函数：
P(X = k) = (λ^k × e^(-λ)) / k!

# 期望：E(X) = λ
# 方差：Var(X) = λ

# 量化应用：极端事件频率
λ = 2（平均每年2次黑天鹅事件）
P(今年发生0次) = e^(-2) ≈ 0.135
P(今年发生3次) = (2^3 × e^(-2)) / 6 ≈ 0.180
```

### 2. 连续分布

#### 均匀分布（Uniform）
**定义**：在[a,b]区间内每个点被选中的概率相等
```python
# 参数：a（下界），b（上界）
# 概率密度函数：
f(x) = 1/(b-a), a ≤ x ≤ b

# 期望：E(X) = (a+b)/2
# 方差：Var(X) = (b-a)²/12

# 量化应用：随机数生成
蒙特卡洛模拟中生成[0,1]随机数
```

#### 正态分布（Normal/Gaussian）
**定义**：最重要的连续分布，呈钟形曲线
```python
# 参数：μ（均值），σ（标准差）
# 概率密度函数：
f(x) = (1/(σ√(2π))) × e^(-((x-μ)²)/(2σ²))

# 期望：E(X) = μ
# 方差：Var(X) = σ²

# 标准正态分布：N(0,1)
Z = (X - μ)/σ

# 重要性质：
68%数据在μ±σ范围内
95%数据在μ±2σ范围内  
99.7%数据在μ±3σ范围内
```

**量化交易中的正态分布假设**
```python
# 经典假设（往往不成立）
股票收益率 ~ N(μ, σ²)

# 实际情况
股票收益率特征：
- 尖峰：中心概率密度更高
- 厚尾：极端事件概率更大
- 偏度：不对称性
- 波动率聚集：高波动后跟高波动

# 改进模型
t分布：更好描述厚尾特征
```

#### 对数正态分布（Log-normal）
**定义**：如果ln(X)服从正态分布，则X服从对数正态分布
```python
# 如果 ln(X) ~ N(μ, σ²)，则 X ~ LogNormal(μ, σ²)

# 期望：E(X) = e^(μ + σ²/2)
# 方差：Var(X) = (e^(σ²) - 1) × e^(2μ + σ²)

# 量化应用：股价建模
股价不能为负，适合用对数正态分布
几何布朗运动模型基础
```

#### 指数分布（Exponential）
**定义**：描述事件间隔时间的分布
```python
# 参数：λ（率参数）
# 概率密度函数：
f(x) = λe^(-λx), x ≥ 0

# 期望：E(X) = 1/λ
# 方差：Var(X) = 1/λ²

# 量化应用：
- 交易信号间隔时间
- 市场崩盘间隔
- 止损被触发的等待时间
```

## 🧠 贝叶斯定理

### 贝叶斯公式

```python
# 基本形式
P(H|E) = P(E|H) × P(H) / P(E)

# 术语解释
P(H|E): 后验概率（给定证据E时，假设H的概率）
P(E|H): 似然（假设H成立时，观察到证据E的概率）  
P(H): 先验概率（不考虑证据E时，假设H的概率）
P(E): 边际概率（证据E发生的总概率）

# 扩展形式（全概率公式）
P(H|E) = P(E|H) × P(H) / [Σ P(E|Hi) × P(Hi)]
```

### 贝叶斯思维在量化交易中的应用

#### 1. 策略信号更新

```python
# 场景：技术指标发出买入信号
先验信息：
- 历史上该股票上涨概率：P(涨) = 0.5
- 历史上该股票下跌概率：P(跌) = 0.5

新证据：技术指标显示买入信号
- 股票上涨时出现买入信号：P(买入信号|涨) = 0.8
- 股票下跌时出现买入信号：P(买入信号|跌) = 0.3

计算后验概率：
P(涨|买入信号) = P(买入信号|涨) × P(涨) / P(买入信号)

其中：P(买入信号) = P(买入信号|涨) × P(涨) + P(买入信号|跌) × P(跌)
                = 0.8 × 0.5 + 0.3 × 0.5 = 0.55

因此：P(涨|买入信号) = (0.8 × 0.5) / 0.55 ≈ 0.727

结论：有了技术信号后，上涨概率从50%提升到72.7%
```

#### 2. 基本面分析更新

```python
# 场景：财报发布后更新股票评级
先验概率（基于历史PE）：
P(被低估) = 0.3
P(合理估值) = 0.5  
P(被高估) = 0.2

新证据：Q3业绩超预期
P(超预期|被低估) = 0.6
P(超预期|合理估值) = 0.3
P(超预期|被高估) = 0.1

# 使用贝叶斯更新
P(被低估|超预期) = (0.6 × 0.3) / P(超预期)
其中 P(超预期) = 0.6×0.3 + 0.3×0.5 + 0.1×0.2 = 0.35

P(被低估|超预期) = 0.18 / 0.35 ≈ 0.51

结论：业绩超预期后，被低估概率从30%提升到51%
```

#### 3. 风险管理中的动态调整

```python
# 场景：市场环境判断
市场状态：牛市、震荡、熊市

先验概率（基于历史统计）：
P(牛市) = 0.3, P(震荡) = 0.5, P(熊市) = 0.2

新证据：北向资金连续5天净流出
P(资金流出|牛市) = 0.2
P(资金流出|震荡) = 0.4  
P(资金流出|熊市) = 0.8

# 计算后验概率
P(熊市|资金流出) = (0.8 × 0.2) / P(资金流出)
= 0.16 / (0.2×0.3 + 0.4×0.5 + 0.8×0.2)
= 0.16 / 0.42 ≈ 0.38

# 策略调整
if P(熊市|新证据) > 0.35:
    reduce_position()  # 降低仓位
    increase_cash()    # 增加现金
```

### 贝叶斯推理的优势

1. **动态更新**：随着新信息不断修正判断
2. **融合多源信息**：整合技术面、基本面、宏观面
3. **量化不确定性**：给出概率而非绝对判断  
4. **避免过度自信**：考虑先验信息的重要性

## 📐 期望值与方差

### 期望值（Expected Value）

**定义与计算**
```python
# 离散随机变量
E(X) = Σ xi × P(X = xi)

# 连续随机变量  
E(X) = ∫ x × f(x) dx

# 期望的性质
E(aX + b) = aE(X) + b
E(X + Y) = E(X) + E(Y)  # 总是成立
E(XY) = E(X)E(Y)        # 仅当X,Y独立时成立
```

**量化交易中的期望值**
```python
# 单次交易期望收益
交易结果 = {
    "盈利": {"概率": 0.6, "收益": 100},
    "亏损": {"概率": 0.4, "收益": -80}
}

期望收益 = 0.6 × 100 + 0.4 × (-80) = 60 - 32 = 28

# 解释：平均每次交易期望赚28元

# 凯利公式应用
f* = (bp - q) / b
其中：
b = 盈利时的赔率 = 100/80 = 1.25
p = 胜率 = 0.6
q = 败率 = 0.4

f* = (1.25 × 0.6 - 0.4) / 1.25 = 0.24

# 建议投注比例：24%的资金
```

### 方差与标准差（Variance & Standard Deviation）

**定义与计算**
```python
# 方差定义
Var(X) = E[(X - E(X))²] = E(X²) - [E(X)]²

# 标准差
σ(X) = √Var(X)

# 方差的性质
Var(aX + b) = a²Var(X)
Var(X + Y) = Var(X) + Var(Y) + 2Cov(X,Y)

# 独立时：Var(X + Y) = Var(X) + Var(Y)
```

**量化交易中的方差应用**
```python
# 组合风险计算
假设两只股票：
股票A：期望收益10%，波动率20%
股票B：期望收益8%，波动率15%
相关系数ρ = 0.3

# 等权重组合（50%-50%）
组合期望收益 = 0.5 × 10% + 0.5 × 8% = 9%

组合方差 = (0.5)²×(20%)² + (0.5)²×(15%)² + 2×0.5×0.5×0.3×20%×15%
        = 0.25×400 + 0.25×225 + 0.25×0.3×300
        = 100 + 56.25 + 22.5 = 178.75

组合波动率 = √178.75 ≈ 13.37%

# 分散化效应：单独持有A或B风险更高，组合降低了风险
```

## 🎲 大数定律与中心极限定理

### 大数定律（Law of Large Numbers）

**弱大数定律**
```python
# 样本均值趋向于真实均值
当n → ∞时，X̄n → μ（依概率收敛）

# 量化交易含义
如果策略的理论期望收益是正的：
- 短期可能亏损（随机性）
- 长期必然盈利（大数定律）

# 实际应用
某策略单次交易期望收益 = +0.5%
进行1000次交易后：
累计收益接近 1000 × 0.5% = 5%
```

**强大数定律**
```python
# 几乎必然收敛
当n → ∞时，X̄n → μ（几乎必然收敛）

# 实际意义
只要策略具有正期望，坚持下去几乎一定能盈利
关键是要有足够的交易次数和资金
```

### 中心极限定理（Central Limit Theorem）

**定理内容**
```python
# 设X1, X2, ..., Xn是独立同分布，均值μ，方差σ²
# 则样本均值的标准化：

Zn = (X̄n - μ) / (σ/√n) → N(0,1)  当n→∞

# 重要结论：
无论原分布是什么形状，样本均值都趋向正态分布
```

**量化交易中的应用**
```python
# 1. 策略收益率分布
即使单次交易收益不是正态分布
但多次交易的平均收益趋向正态分布

# 2. 风险度量（VaR计算）
假设日收益率独立，年收益率近似正态
可以用正态分布计算风险指标

# 3. 假设检验
检验策略是否真的有效：
H0: 策略无效（期望收益 = 0）
H1: 策略有效（期望收益 > 0）

使用中心极限定理构造检验统计量
```

**实际计算示例**
```python
# 某策略100次交易记录
单次收益均值 = 0.8%
单次收益标准差 = 5%

# 根据中心极限定理
平均收益的标准误差 = 5% / √100 = 0.5%

# 95%置信区间
置信区间 = 0.8% ± 1.96 × 0.5% = [0.8% ± 0.98%] = [-0.18%, 1.78%]

# 由于包含0，无法断定策略显著有效
```

## 📈 量化交易中的概率应用

### 1. 风险价值（VaR）计算

**参数法（假设正态分布）**
```python
# 给定置信水平α，计算VaR
VaR_α = μ - z_α × σ

# 例子：日VaR计算
日平均收益 = 0.05%
日波动率 = 1.5%
置信水平 = 95%（z_0.05 = 1.645）

VaR_95% = 0.05% - 1.645 × 1.5% = -2.42%

# 解释：95%概率下，日损失不超过2.42%
```

**历史模拟法**
```python
# 使用历史数据直接计算
历史收益率序列：[-3.2%, -2.1%, -1.8%, 0.1%, 0.5%, 1.2%, 2.1%, ...]
排序后取第5%分位数作为VaR_95%

import numpy as np
returns = np.array([-0.032, -0.021, -0.018, ...])
VaR_95 = np.percentile(returns, 5)
```

**蒙特卡洛模拟法**
```python
# 生成大量随机路径
import numpy as np

def monte_carlo_var(mu, sigma, days, n_simulations=10000):
    """
    蒙特卡洛模拟计算VaR
    """
    # 生成随机收益率路径
    random_returns = np.random.normal(mu, sigma, (n_simulations, days))
    
    # 计算每条路径的累计收益
    cumulative_returns = np.prod(1 + random_returns, axis=1) - 1
    
    # 计算VaR
    var_95 = np.percentile(cumulative_returns, 5)
    return var_95

# 计算10日VaR
var_10d = monte_carlo_var(0.0005, 0.015, 10)
```

### 2. 期权定价中的概率

**Black-Scholes模型假设**
```python
# 股价遵循几何布朗运动
dS = μS dt + σS dW

# 其中W是维纳过程（布朗运动）
# 股价对数收益率服从正态分布：
ln(St/S0) ~ N((μ - σ²/2)t, σ²t)

# 期权定价公式中的概率项
d1 = (ln(S/K) + (r + σ²/2)T) / (σ√T)
d2 = d1 - σ√T

Call = S×N(d1) - K×e^(-rT)×N(d2)

# N(d2) = 股价超过执行价的风险中性概率
```

### 3. 策略组合优化

**马科维茨投资组合理论**
```python
# 目标：在给定风险水平下最大化收益
# 或在给定收益水平下最小化风险

# 数学表达：
minimize: w^T Σ w  (组合方差)
subject to: w^T μ = r  (目标收益)
           Σ wi = 1   (权重和为1)

其中：
w = 权重向量
Σ = 协方差矩阵  
μ = 期望收益向量
r = 目标收益率
```

**风险平价模型**
```python
# 每个资产对组合风险的贡献相等
风险贡献i = wi × (Σw)i / (w^T Σ w)

# 目标：所有资产的风险贡献相等
RC1 = RC2 = ... = RCn = 1/n
```

成功收益 = (100 - 95) / 95 = 5.26%
失败亏损 = 假设跌到90，(90 - 95) / 95 = -5.26%

期望收益 = 0.58 × 5.26% + 0.42 × (-5.26%) ≈ 0.84%
年化收益 = 0.84% × 4 = 3.36%

# 决策：如果无风险利率<3.36%，则值得投资

    def bayes_theorem_demo(self):
        """贝叶斯定理演示"""
        print("\n=== 贝叶斯定理应用演示 ===")
        
        # 场景：技术指标预测股价
        prior_up = 0.5      # 先验：股价上涨概率
        prior_down = 0.5    # 先验：股价下跌概率

                         indicator_given_down * prior_down)

        posterior_up = (indicator_given_up * prior_up) / prob_indicator
        posterior_down = (indicator_given_down * prior_down) / prob_indicator

        print(f"指标准确性 - 下跌时买入信号: {indicator_given_down:.1%}")
        print(f"后验概率 - 买入信号后上涨: {posterior_up:.1%}")
        print(f"后验概率 - 买入信号后下跌: {posterior_down:.1%}")
        
        # 可视化
        categories = ['先验概率', '后验概率']
        up_probs = [prior_up, posterior_up]

    b = abs(avg_win / avg_loss)  # 赔率

    daily_return_std = 0.015
    confidence_level = 0.95
    
    # 参数法VaR
    z_score = stats.norm.ppf(1 - confidence_level)
    var_parametric = portfolio_value * (daily_return_mean + z_score * daily_return_std)
    
    print(f"组合价值: {portfolio_value:,}")
    print(f"日均收益: {daily_return_mean:.2%}")
    print(f"日波动率: {daily_return_std:.1%}")
    print(f"95% VaR: {-var_parametric:,.0f} 元")
    
    # 4. 连续亏损概率
    print("\n4. 连续亏损概率分析")

def probability_exercises():

    P_signal_given_down = 0.3
    P_up = 0.5
    P_down = 0.5
    
    P_signal = P_signal_given_up * P_up + P_signal_given_down * P_down
    P_up_given_signal = (P_signal_given_up * P_up) / P_signal
    
    print(f"答案：{P_up_given_signal:.2%}\n")
    
    # 练习2：期望收益计算
    print("练习2：期望收益与风险")
    print("某交易策略：")
    print("- 70%概率盈利5%")
    print("- 30%概率亏损8%")
    print("计算期望收益和风险")
    
    returns = [0.05, -0.08]
    probs = [0.7, 0.3]
    
    expected_return = sum(r * p for r, p in zip(returns, probs))
    variance = sum(p * (r - expected_return)**2 for r, p in zip(probs, returns))
    std_dev = np.sqrt(variance)
    
    print(f"期望收益：{expected_return:.2%}")
    print(f"风险（标准差）：{std_dev:.2%}\n")
    
    # 练习3：组合优化
    print("练习3：两资产组合优化")
    print("资产A：期望收益12%，波动率20%")
    print("资产B：期望收益8%，波动率15%") 
    print("相关系数0.3，求最小风险组合的权重")
    
    mu_a, mu_b = 0.12, 0.08
    sigma_a, sigma_b = 0.20, 0.15
    rho = 0.3
    
    # 最小方差组合权重
    numerator = sigma_b**2 - rho * sigma_a * sigma_b
    denominator = sigma_a**2 + sigma_b**2 - 2 * rho * sigma_a * sigma_b
    w_a = numerator / denominator
    w_b = 1 - w_a
    
    print(f"资产A权重：{w_a:.2%}")
    print(f"资产B权重：{w_b:.2%}")
    
    # 计算组合特性
    portfolio_return = w_a * mu_a + w_b * mu_b
    portfolio_variance = (w_a**2 * sigma_a**2 + w_b**2 * sigma_b**2 + 
                         2 * w_a * w_b * rho * sigma_a * sigma_b)
    portfolio_std = np.sqrt(portfolio_variance)
    
    print(f"组合期望收益：{portfolio_return:.2%}")
    print(f"组合风险：{portfolio_std:.2%}")
