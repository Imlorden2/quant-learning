**概率分布**是描述**随机变量**取各个值的概率规律的数学函数或表格
随机变量：实验结果的数学表示

根据实验类型的不同，随机变量分为离散随机变量和连续随机变量。
- 离散随机变量的概率分布用 PMF 表示
- 连续随机变量的概率分布用 PDF 表示

#### PMF : Probability Mass Function(概率质量函数)
- 定义： P(X = x)
- 直接给出 X 取某个值的概率
$$P(X=k) = \frac{1}{6}, \quad k \in {1,2,3,4,5,6}$$

#### PDF : Probability Density Function(概率密度函数)
- 定义： f(x)
- 本身不是概率，对区间积分才得概率
$$P(a \leq X \leq b) = \int_a^b f(x),dx$$


### Python 计算方法

```python
from scipy import stats

# PMF：P(X = k)
stats.binom.pmf(6, 10, 0.6)       # P(X = 6)

# CDF：P(X ≤ k)
stats.binom.cdf(6, 10, 0.6)       # P(X ≤ 6)

# SF：P(X > k)
stats.binom.sf(6, 10, 0.6)        # P(X > 6) = P(X ≥ 7)
```
