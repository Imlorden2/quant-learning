## 🔥 核心术语 (必须掌握)

### A-D
| 术语 | 英文 | 定义 | 计算公式 |
|-----|------|------|---------|
| **阿尔法** | Alpha | 超越基准的收益 | α = 策略收益 - 基准收益 |
| **贝塔** | Beta | 相对市场敏感度 | β = Cov(策略,市场)/Var(市场) |
| **波动率** | Volatility | 收益率标准差 | σ = √(Σ(Ri-R̄)²/(n-1)) |
| **布林带** | Bollinger Bands | 价格通道指标 | 上轨=MA+2σ, 下轨=MA-2σ |
| **做多** | Long | 买入看涨 | 盈利 = 卖价 - 买价 |
| **做空** | Short | 卖出看跌 | 盈利 = 卖价 - 买价 |
| **对冲** | Hedge | 降低风险操作 | - |
| **动量** | Momentum | 趋势延续性 | 过去N期收益率 |

### E-L  
| 术语 | 英文 | 定义 | 计算公式 |
|-----|------|------|---------|
| **夏普比率** | Sharpe Ratio | 风险调整收益 | (年化收益-无风险利率)/年化波动率 |
| **杠杆** | Leverage | 借钱放大交易 | 总资产/自有资金 |
| **基差** | Basis | 现货期货价差 | 现货价格 - 期货价格 |
| **均值回归** | Mean Reversion | 价格回归平均 | - |
| **流动性** | Liquidity | 买卖便利程度 | 买卖价差、成交量等 |

### M-S
| 术语 | 英文 | 定义 | 计算公式 |
|-----|------|------|---------|
| **MACD** | MACD | 趋势指标 | DIF=EMA12-EMA26, DEA=EMA(DIF,9) |
| **最大回撤** | Max Drawdown | 最大损失幅度 | (峰值-谷值)/峰值×100% |
| **移动平均** | Moving Average | 平滑价格指标 | MA(n) = (P1+P2+...+Pn)/n |
| **RSI** | RSI | 相对强弱指标 | 100-100/(1+RS), RS=涨幅均值/跌幅均值 |
| **滑点** | Slippage | 成交价差异 | 实际价格 - 预期价格 |

### T-Z
| 术语 | 英文 | 定义 | 计算公式 |
|-----|------|------|---------|
| **套利** | Arbitrage | 无风险获利 | - |
| **仓位** | Position | 持有头寸 | 持仓金额/总资产 |
| **胜率** | Win Rate | 盈利交易占比 | 盈利次数/总交易次数×100% |
| **盈亏比** | Risk Reward | 盈亏比例 | 平均盈利/平均亏损 |
| **因子** | Factor | 影响收益变量 | - |
| **止损** | Stop Loss | 亏损时平仓 | - |

## ⚠️ 风险管理公式

### 仓位管理
```
凯利公式: f* = (bp-q)/b
f* = 最优下注比例
b = 赔率 (盈利/本金)
p = 胜率
q = 败率 (1-p)

等风险法: Wi = σp/(σi × N)
Wi = 资产i权重
σp = 组合目标波动率
σi = 资产i波动率  
N = 资产数量
```

### 风险指标
```
VaR (95%): 
历史法 = 收益率序列第5%分位数

夏普比率:
Sharpe = (Rp - Rf) / σp
Rp = 组合收益
Rf = 无风险收益
σp = 组合波动率

最大回撤:
MDD = max[(Peakt - Valleyt) / Peakt]
```

## 💰 收益计算

### 基础收益
```python
# 简单收益率
R = (P1 - P0) / P0

# 对数收益率  
r = ln(P1/P0)

# 年化收益率
Ra = (1 + R)^(365/days) - 1

# 复合年化增长率(CAGR)
CAGR = (Vend/Vbegin)^(1/years) - 1
```

### 组合收益
```python
# 加权收益
Rp = Σ(wi × Ri)

# 几何平均收益
Rg = (∏(1 + Ri))^(1/n) - 1

# 超额收益
Excess = Rp - Rb
```
